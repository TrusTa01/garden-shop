## Технические требования

### Используемые библиотеки (обязательные)
- **React** (18.x или выше) - основная библиотека UI
- **Redux Toolkit (RTK)** - управление состоянием приложения
- **React Router DOM** (v6) - маршрутизация
- **React Hook Form** - управление формами и валидация
